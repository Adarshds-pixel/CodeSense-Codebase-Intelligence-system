# Architecture Notes

## Suggested High-Level Structure

- `frontend/` for React.js and Tailwind CSS UI
- `backend/` for FastAPI APIs and analysis services
- `data/` for uploads, repository snapshots, and vector storage
- `docs/` for architecture, API, database, and planning notes
- `docker/` for deployment scaffolding

## Planned Modules

- Authentication Module
- Repository Upload Module
- GitHub Integration Module
- Code Analysis Engine
- AI Explanation Module
- Search and Retrieval Module
- Dashboard and Reporting Module
