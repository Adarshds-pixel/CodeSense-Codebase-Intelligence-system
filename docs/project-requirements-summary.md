# Project Requirements Summary

## Problem

Developers and students need a faster way to understand large codebases without reading every file manually.

## Core Objectives

- Analyze project repositories automatically
- Generate summaries for files, folders, and modules
- Explain code in simple language
- Visualize architecture and dependencies
- Integrate with GitHub repositories
- Support semantic search across the codebase

## Inputs

- GitHub repository links
- ZIP file uploads
- Project files

## Outputs

- Project summary
- File explanations
- Folder analysis
- Dependency graph
- Documentation
